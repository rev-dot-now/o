## Installation from Source

Getting started with **o** is a breeze. Just clone the repository and install Bun, and you’re ready to dive into the world of file management like never before. 

- Clone the repository:
  ```bash
  git clone https://github.com/rev-dot-now/o.git
  ```
- Install Bun:
  [Documentation](https://bun.sh/docs/installation)

## Run Locally

Ready to see **o** in action? Running it locally is straightforward. Fire up your terminal and run the command below. You’ll be managing files and directories in no time. 

```bash
bun run dev
```

## Build

Want to take it a step further? Building the **o** command is your next move. This will create the `o` command in the `dist/` folder at the root of the project. Once built, you can use it anywhere on your system. 

```bash
bun run build
```

## Usage Examples

Now that you have **o** up and running, let’s check out some examples. With **o**, you can create files, manage directories, and so much more. 

```bash
# Example command
o create myfile.txt
```
