# **o** 🌀  

**The Everything CLI**: *a tool that does everything, probably too much.* Inspired by the Everything Bagel, **o** gazes into the chaos and lets you run shell commands with the confidence of someone who doesn’t fear entropy—because you *are* entropy.  

## **What is "o"?**  
**o** is a CLI tool for hackers, tinkerers, and void-gazers. It can:  
- Run your scripts.  
- Access your file system (sometimes more enthusiastically than you'd like).  
- Embrace chaos and maybe summon something... existential.  

Essentially, it's like if you asked a terminal to be philosophical *and* destructive at the same time.
