# **o** 🌀  

Short for "ouroboros" 🐍, **o** is a simple, self-referential CLI tool for
generating and managing files and directories.  It was born out of experiments
by Hans Oksendahl in creating CLI tools that could self-modify and extend their
own functionality.

## Installation from Source

- Clone the repository
  ```bash
  git clone https://github.com/rev-dot-now/o.git
  ```
- Install Bun
  [Documentation](https://bun.sh/docs/installation)

## Run Locally

```bash
bun run dev
```

## Build

This will create the `o` command in the `dist/` folder at the root of the
project.

```bash
bun run build
```
